## CONFIGURATION AND SET-UP

### COMPANY SETTINGS

- **Company Information**
  - View and edit organization information, including:
    - Organization Name
    - Primary Contact
    - Default Locale
    - Default Language
    - Default Timezone

- **Business Hours**
  - Set organizational business hours, with options to:
    - Define specific time ranges for each day of the week
    - Create multiple sets of business hours for flexibility
  - Useful for managing service hours and availability.

- **Holiday**
  - Define company holidays observed by the organization.
  - Holidays suspend standard business hours.

- **Fiscal Year**
  - Define the start and end dates for the fiscal year.
  - Supports standard fiscal year functionality, allowing you to:
    - Choose a starting and ending month for the fiscal year
    - Customize the fiscal year to meet financial reporting needs.

- **Language Settings**
  - Enable multiple languages to accommodate users with different language preferences.

- **Users**
  - Configure user roles and permissions:
    - Set roles for each user
    - Define what each user can access and view
    - Option to "Generate new password and notify the user immediately" when needed.
